## はじめての CSS

まずは、CSS を用いて `こんにちは、世界！` の文字を赤くしてみましょう。

次の HTML のコードと CSS のコードを JSFiddle に貼り付けてください。

```html title="HTML のコード"
<!doctype html>
<html lang="ja">
  <head>
    <meta charset="utf-8" />
    <title>タイトル</title>
  </head>
  <body>
    <p id="greeting">こんにちは、世界！</p>
  </body>
</html>
```

```css title="CSS のコード"
#greeting {
  color: red;
}
```

保存すると、画面に `こんにちは、世界！` と表示され、文字色が赤くなっているはずです。

## CSS の仕組み

先ほどのコードを詳しく見ていきます。

### `id` 属性の指定

CSS を記述する際には、まずスタイルを適用する対象となる HTML 要素を指定する必要があります。
このために、スタイルを適用する対象となる HTML 要素に `id` 属性を指定します。

先ほどの例では、HTML のコードで `strong` 要素に対して、`greeting` という `id` 属性を指定しました。

```html
<p id="greeting">こんにちは、世界！</p>
```

### セレクタ

次に、CSS でどの HTML 要素に対してスタイルを適用するのかを指定します。
ここで、使用するのが**セレクタ**です。

先ほどの例では、CSS のコードで `#greeting` のように書くことで、`greeting` という `id` 属性を持つ HTML 要素に対してスタイルを適用するように指定しています。

```css
#greeting {
  color: red;
}
```

### プロパティとプロパティ値

次に、指定した HTML 要素に対して、どのようなスタイルを適用するのかを記述します。

```css
#greeting {
  color: red;
}
```

先ほどの例では、`color` という<Term type="cssProperty">**プロパティ**</Term>に、`red` という**プロパティ値**を指定することで、文字色を赤色にしています。

{/* prettier-ignore */}
<Term type="cssProperty">プロパティ</Term>はスタイルの種類を表し、プロパティ値はプロパティに指定できる値を表します。

## 複数のプロパティ

```css
#greeting {
  color: red;
  font-size: 30px;
}
```

複数の<Term type="cssProperty">プロパティ</Term>を指定する場合には、上のように `プロパティ: プロパティ値;` の組を並べて記述します。
ここでは、文字サイズを `30px` の大きさにしています。

CSS の<Term type="cssProperty">プロパティ</Term>には `color` (文字色) や `font-size` (文字サイズ) だけでなく、`background-color` (背景色)、`text-decoration` (文字装飾) 等、数えきれないほどの種類が定義されています。
様々なプロパティを調べてみてください。

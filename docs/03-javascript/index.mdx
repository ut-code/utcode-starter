## はじめての JavaScript

<Term>JavaScript</Term> を用いて画面に `こんにちは、世界！` と表示してみましょう。

次の JavaScript のプログラムを JSFiddle に貼り付けてください。

```javascript
document.write("こんにちは、世界！");
```

保存すると、画面に `こんにちは、世界！` と表示されるはずです。

## JavaScript の基本文法

`document.write` は画面に出力するための命令です。

```javascript
document.write("こんにちは、世界1");
document.write("こんにちは、世界2");
document.write("こんにちは、世界3");
```

## コメント

`//` から行末までの部分は**コメント**とみなされ、プログラムの実行に影響を与えません。
この教材内でもプログラムの意味や実行結果を説明するのに利用していきます。

```javascript
// この部分はコメントです。
document.write("こんにちは、世界1"); // この部分もコメントです。
// 次の行はコメントになっているため、プログラムの実行に影響を与えません。
// document.write("こんにちは、世界2");
document.write("こんにちは、世界3");
```

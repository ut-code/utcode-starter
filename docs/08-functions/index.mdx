## 関数の基本

数学における関数とは異なり、プログラミング言語における**関数**は、基本的には<Term>文</Term>のまとまりに名前を付けたものです。

次のプログラムを実行すると、`こんにちは、世界！こんにちは、世界！` と表示されます。

```javascript showLineNumbers
// 関数を定義しておけば、
function greet() {
  document.write("こんにちは、世界！");
}

// 後から何度も呼び出すことができます。
greet(); // こんにちは、世界！
greet(); // こんにちは、世界！
```

2 行目から 4 行目で<Term>関数</Term>を定義しています。
ここでは、`{` と `}` で囲まれた<Term>文</Term>のまとまりに `greet` という名前を付けています。

7 行目と 8 行目で、関数 `greet` を 2 回呼び出しています。
`greet();` のように書くことで、<Term>関数</Term>を呼び出すことができます。
これにより、`こんにちは、世界！` が 2 回表示されます。

### 確認問題

`はじめまして。` と自己紹介をする関数 `introduceSelf` を定義してください。
また、呼び出してください。

```javascript
function introduceSelf() {
  // ここに関数が行う処理を書いてください。
}

// ここで関数を呼び出してください。
```

<Answer title="自己紹介">

```javascript

  document.write("はじめまして。");
```

## 引数

関数の振る舞いを呼び出し時に変えるため、関数に**引数**を与えることができます。<Term>引数</Term>にはどんな<Term>値</Term>でも指定できます。

次のプログラムを実行すると、`こんにちは、田中さん！こんにちは、鈴木さん！` と表示されます。

```javascript showLineNumbers
function greet(name) {
  document.write("こんにちは、" + name + "さん！");
}

greet("田中"); // こんにちは、田中さん！
greet("鈴木"); // こんにちは、鈴木さん！
```

`greet` 関数には、`name` という<Term>引数</Term>が定義されています。

`greet("田中");` のようにして関数を呼び出すと、`"田中"` という<Term>値</Term>が<Term>引数</Term>として指定されます。

この<Term>値</Term>は、関数内では `name` という<Term>引数</Term>として扱われます。

関数の中では<Term>引数</Term>は<Term>変数</Term>のように振る舞うため、`document.write("こんにちは、" + name + "さん！");` を実行すると、`こんにちは、田中さん！` が表示されます。

### 確認問題

名前を表す引数 `name` を受け取り、`はじめまして。私の名前は〇〇です。` と自己紹介をする関数 `introduceSelf` を定義してください。
また、引数に `"田中"` を与えて呼び出してください。

```javascript
function introduceSelf(name) {
  // ここに関数が行う処理を書いてください。
}

// ここで関数を呼び出してください。
```

<Answer title="自己紹介">

```javascript

  document.write("はじめまして。私の名前は" + name + "です。");
}

introduceSelf("田中");
```

## 複数の引数

複数の<Term>引数</Term>を受け取る関数を定義することもできます。

次のプログラムを実行すると、`おはよう、田中さん！こんにちは、鈴木さん！` と表示されます。

```javascript
function greet(greetingMessage, name) {
  document.write(greetingMessage + "、" + name + "さん！");
}

greet("おはよう", "田中");
greet("こんにちは", "鈴木");
```

`greet("おはよう", "田中");` のようにして関数を呼び出すと、`"おはよう"` が `greetingMessage` という<Term>引数</Term>に、`"田中"` が `name` という<Term>引数</Term>にそれぞれ指定されます。

`name` と `age` という引数を受け取り、`はじめまして。私の名前は〇〇です。〇〇歳です。` と自己紹介をする関数 `introduceSelf` を定義してください。
また、引数に `"田中"` と `12` を与えて呼び出してください。

```javascript
// ここで関数を定義してください。

// ここで関数を呼び出してください。
```

<Answer title="自己紹介">


## 戻り値

関数内から関数の呼び出し元に<Term>値</Term>を返すことができます。
また、この<Term>値</Term>を**戻り値**と呼びます。

次のプログラムを実行すると、`1と2の和は3です。` と表示されます。

```javascript
function add(a, b) {
  let sum = a + b;
  return sum;
}

let answer = add(1, 2);
document.write("1と2の和は" + answer + "です。"); // 1と2の和は3です。
```

`return sum` のようにすることで、そこで関数の実行を停止し、関数の呼び出し元に変数 `sum` の<Term>値</Term>を<Term>戻り値</Term>として返すことができます。そのため、`add(1, 2)` のように関数を呼び出すと、引数 `a` は `1`、引数 `b` は `2`となり、`sum` が `3` となるので、`3` が<Term>戻り値</Term>となります。

このため、`add(1, 2)` は戻り値として返ってきた `3` になり、この<Term>値</Term>が `answer` に代入されます。
これにより、`1と2の和は3です。` と表示されます。

## 演習問題

### 問題 1

あるテーマパークの入場料金は、12 歳未満なら 500 円、12 歳以上なら 1000 円です。

このテーマパークの入場料金を計算する関数を作ってみましょう。

年齢を表す引数 `age` を受け取り、12 未満なら `500`、12 以上なら `1000` を<Term>戻り値</Term>として返す関数 `calculateTicketPrice` を定義してください。
また、引数に `12` を与えて呼び出してください。

```javascript
function calculateTicketPrice(age) {
  if (/* ここに条件を書いてください。 */) {
    return /* ここに12歳未満の料金を書いてください。 */;
  } else {
    return /* ここに12歳以上の料金を書いてください。 */;
  }
}

document.write("入場料金は" + calculateTicketPrice(12) + "円です。");
```

<Answer title="入場料金">

```javascript
function calculateTicketPrice(age) {
  if (age < 12) {
    return 500;
  } else {
    return 1000;
  }
}

document.write("入場料金は" + calculateTicketPrice(12) + "円です。");
```
